# Iteration

* [陳鍾誠 : 迭代法 Iterative (JavaScript)](https://misavo.com/blog/%E9%99%B3%E9%8D%BE%E8%AA%A0/%E6%9B%B8%E7%B1%8D/%E6%BC%94%E7%AE%97%E6%B3%95/04-iterative)

求 x = f(x) 的解

方法：

1. 隨便選一個 x=x0
2. x1 = f(x0)
   x2 = f(x1)
   ....
   xn = f(x[n-1])

3. 一但收斂了 (x[n] = x[n-1]), 
   就代表 xn = f(xn)
   就輸出結果。
